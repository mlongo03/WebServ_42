BEHAVIOUR WITH POSTMAN

Create a file and put some test inside it

we modifiy the redirection to /new-page/file.txt

we change the method to DELETE with postman

if we set the redirection to 301/302/303

then postman will redirect to /new-page/file

and return a GET

the webserver show the file


if we set the redirection to 307 or 308

postman will redirect to /new-page/file.txt

and perform the DELETE on that resource

returning a 204 No content


so 301 302 303 always change the method to GET

307 308 keep the original method
